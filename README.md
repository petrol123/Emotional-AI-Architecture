# Emotional Weight Engine (EWE) – Reference Implementation

## Overview
This repository contains a minimal, deterministic reference implementation of the Emotional Weight Engine (EWE), a core component of the Socio-Emotional Communication Engine (SECE) architecture.

The purpose of this implementation is not to provide a production-ready system, but to demonstrate the internal coherence, explainability, and evaluability of emotional weighting as a standalone mechanism.

## Design Principles
- Extensible emotional constructs (not fixed emotion axes)
- Deterministic behavior for evaluation and reproducibility
- Explicit separation of emotional intensity and qualitative significance (qualia)
- Explainable state transitions
- Compatibility with Portable Knowledge Base (PKB) snapshots

## What This Is
- A conceptual and executable reference
- A behavioral model suitable for inspection and evaluation
- A foundation for further research or implementation

## What This Is Not
- A dialogue system
- A sentiment classifier
- A trained machine learning model
- A clinical decision system

## Usage
The implementation is contained in a single Python file (`ewe.py`) and has no external dependencies.

Example usage scenarios are described within the accompanying documentation and Zenodo record.

## Status
This implementation represents a foundational reference version intended to support future integration, evaluation, and collaboration.
